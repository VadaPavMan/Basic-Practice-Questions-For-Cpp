**1. Vector basic use case:**
Create a vector of an integer array, and define its size. then take numbers input from the user, store in array and display the numbers of array, define a function for display.

Output Example: 
```

Enter the number: 1
Enter the number: 2
Enter the number: 3
Enter the number: 4

1 2 3 4 5
```
